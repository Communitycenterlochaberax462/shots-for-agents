## 🧭 What this app does

shots-for-agents is a small screen capture app for people who work with AI agents. It takes one clear screenshot at a time, so you can share what is on your screen without extra steps.

Use it when you need to:
- capture a chat window
- show an error message
- share part of a webpage
- send a clean screen image to an AI tool
- keep a simple record of what you saw

It is built as a Mac menu bar utility, so it stays out of the way until you need it.

## 🖥️ What you need

This app is made for Mac computers.

You need:
- macOS 12 or later
- a mouse or trackpad
- permission to take screenshots
- a few megabytes of free disk space

If you plan to use it with AI tools, you may also want:
- a folder for saved screenshots
- a fast way to copy images to the clipboard
- a cloud drive or chat app for sharing

## 📸 How to use it

1. Click the app icon in the menu bar
2. Choose the screenshot action
3. Select the area or window you want to capture
4. Save the image or copy it to the clipboard
5. Paste it into your AI chat, email, or notes app

For a smooth workflow:
- use one screenshot for one task
- keep the image clear and cropped
- capture only what the agent needs
- avoid full-screen clutter when you can

## 🧩 Troubleshooting

If the app does not open:
- make sure you downloaded the latest file
- try opening it from Applications
- check that macOS did not block it

If screenshots are blank:
- confirm Screen Recording access is on
- close and reopen the app
- try again after granting permission

If the app does not appear in the menu bar:
- look for hidden menu bar icons
- widen the top-right area if your screen is crowded
- restart the app

If hotkeys do not work:
- check Accessibility access
- make sure another app is not using the same key combo
